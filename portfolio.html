Smart Home Automation

Satellite Image Denoising

Developed a system to remove noise from satellite images using Convolutional Neural Networks (CNNs) and denoising autoencoders. Implemented patch-based image processing for better efficiency and accuracy. Trained the model with supervised learning techniques to optimize results. Evaluated the performance using Peak Signal-to-Noise Ratio (PSNR) to ensure high-quality outputs. The project enhances satellite image clarity, making it useful for remote sensing and analysis applications.

Smart Blood Bag Storage

Designed an IoT-based system to monitor and secure blood bags in real-time. Integrated RFID technology for tracking and fingerprint authentication for secure access. Implemented temperature sensors to maintain optimal storage conditions and alert notifications for anomalies. Developed a mobile app interface for monitoring and control. The system improves efficiency, traceability, and safety in blood transfusion services.
